Kirok

An offline-first mobile workout tracker built for serious lifters. Record sets, detect personal records, manage routines, and sync seamlessly across devices.

View on GitHub Learn More

// features

Built for the Gym

⚡

Set Recording

Log exercises, weights, reps, and RPE with a fast, minimal interface designed for between-set use.

🏆

PR Detection

Automatically detects personal records across exercises and celebrates your new bests.

🔄

Routine Management

Create, save, and reuse workout routines. Start any session from a template or go freestyle.

📡

Offline-First

Works without internet using local SQLite storage. Syncs to cloud when you're back online.

⌨

Apple Watch

Log workouts directly from your wrist with Apple Watch and Health integration.

📈

Analytics

Custom SVG charts for volume, frequency, and strength progression over time.
