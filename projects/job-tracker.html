JobTracker

An AI-powered job application tracker that auto-extracts job details from posting URLs. Comes with a Chrome extension for one-click capture from LinkedIn, Indeed, and Glassdoor.

View on GitHub Learn More

Features

Job Search, Simplified

🤖

AI Extraction

Paste a job URL and AI automatically extracts the title, company, location, and description. Supports Claude, OpenAI, and Gemini.

🔌

Chrome Extension

One-click capture from LinkedIn, Indeed, and Glassdoor. Save jobs without leaving the page.

📋

Dashboard

Track all your applications in one place with status updates, notes, and filtering.

📝

Auto-Fill Profiles

Automatically fills your profile URLs on application forms to speed up the process.

💾

Local-First

All data stored locally with PostgreSQL. Your job search data stays on your machine.

🔧

Multi-LLM

Choose your AI provider. Works with Anthropic Claude, OpenAI GPT, and Google Gemini.

Built With

Next.js React 19 TypeScript Prisma PostgreSQL Claude SDK Tailwind CSS Chrome Extension
